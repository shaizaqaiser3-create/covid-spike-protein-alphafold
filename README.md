# covid-spike-protein-alphafold
AI-based protein structure prediction project using AlphaFold to analyze the SARS-CoV-2 spike glycoprotein structure and secondary structural elements.
# Protein Structure Prediction of SARS-CoV-2 Spike Protein Using AlphaFold

This project focuses on predicting and analyzing the 3D structure of the SARS-CoV-2 spike glycoprotein using AlphaFold and bioinformatics tools.

## Tools Used
- UniProt
- PSIPRED
- AlphaFold
- UCSF ChimeraX

## Objectives
- Retrieve protein sequence
- Predict secondary structure
- Generate 3D protein structure
- Visualize and analyze structure

## Results
The spike protein structure was successfully predicted and visualized. The project demonstrates the importance of AI-based protein structure prediction in modern bioinformatics.
